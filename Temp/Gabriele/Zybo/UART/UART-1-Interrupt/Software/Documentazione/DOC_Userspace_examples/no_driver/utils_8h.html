Descrizione dettagliata

La liberia utils definisce delle funzioni comuni alla gestione semplificata di una periferica AXI Lite generica
.

Le funzioni da prendere in considerazione dipendono dal contesto di esecuzione.
Bare metal considera un contesto in cui non è presente il sistema operativo e si è a diretto contatto con l'hardware.
Le altre tipologie di API presenti nella libreria considerano la presenza di un sistema operativo.

void* configure_no_driver ( int  file_descriptor,
void **  vrt_page_addr,
uint32_t  phy_address 
)

Esegue mmap sul file /dev/mem aperto con la seguente configurazione: PROT_READ | PROT_WRITE e MAP_SHARED.

Parametri
file_descriptordescrittore del file /dev/mem aperto
vrt_page_addrpuntatore all'indirizzo virtuale della pagina a cui appartiene l'indirizzo virtuale della periferica memory mapped (parametro di ingresso uscita)
phy_addressindirizzo fisico base della periferica memory mapped
Valori di ritorno
vrt_addrNULL nel caso in cui "mmap" ritorni MAP_FAILED, altrimenti l'indirizzo virtuale base della periferica
